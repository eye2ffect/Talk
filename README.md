**1. 개요**
* **목적:** TCP 통신 기초 및 스레드 제어 원리 파악
* **기술:** C#, Windows Forms, `System.Net.Sockets`

**3. 핵심 구현**
* **스레드 분리:** 네트워크 수신(`ReadLine`) 대기 시 발생하는 UI 프리징을 막기 위해 백그라운드 스레드 적용.
* **UI 동기화:** `Control.Invoke`를 활용하여 통신 스레드에서 UI 스레드로 안전하게 데이터 전달.

**4. 인사이트 및 개선점**
* **동기식 I/O의 한계:** 스레드 점유 문제를 확인하였으며, 게임 환경에 필수적인 비동기(Async) 네트워크 모델의 필요성 체감.
* **패킷 직렬화:** 단순 문자열 전송의 한계를 파악하고, 바이너리 기반의 패킷 구조화 도입 필요성 확인.
